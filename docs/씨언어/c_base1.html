상수, 변수, 자료형, 선행처리기

자료형
프로그램에서 사용하는 자료의 형태를 자료형이라고 합니다. 자료형은 상수와 변수로 구분하여 사용됩니다. C 프로그램은 자료 처리를 위해 수많은 명령어와 자료를 사용하며, 이들 자료의 정확한 처리와 효율적인 활용을 위해 다양한 자료형을 제공합니다.

상수​

  • 항상 고정된 값을 갖는 자료
  • 값이 한번 정해지면 프로그램 도중 그 값을 변경할 수 없는 수

정수형 상수​

  • 10진수, 8진수, 16진수로 표현
구분예예시
10진수10, 20, 300부터 9까지의 숫자를 사용하며, 0으로 시작할 수 없음
8진수012, 034, 0760부터 7까지의 숫자를 사용하며, 숫자 앞에 0을 붙임
16진수0x1A, 0xFF, 0x7B0부터 9까지의 숫자와 A부터 F까지의 문자를 사용하며, 숫자 앞에 0x를 붙임
unsigned 형10U, 20U, 30U부호 없는 정수를 나타내며, U 또는 u를 사용하여 표현
long 형10L, 20L, 30L큰 정수를 나타내며, L 또는 l을 사용하여 표현

실수형 상수​

  • 부동소수점 형 상수
  • double형을 기본 자료형으로 사용
구분예비고
소수형식12.345, 4.19소수점을 사용하여 표현
지수형식1.23e4, 4.19e-3과학적 표기법으로 표현 (e 또는 E 사용)
float 형1.23f, 4.19F숫자 뒤에 f,F를 사용하여 표현
long double 형1.23L, 4.19l숫자 뒤에 l이나 L을 사용하여 표현

문자형 상수​

  • 단일 인용부호(’’)로 묶여 있는 1개의 영문자나 숫자문자
  • 내부적으로는 해당문자의 ASCll 코드값이 사용
    • 예를 들어 ‘A’는 내부적으로 65(ASCll 코드값)가 사용된다.
  • Escape 문자: 키보드에 나타나 있지 않은 문자.
Escape 문자기능
\a경고음(alert) 출력
\b백스페이스(backspace)
\f페이지 넘김(form feed)
\n줄 바꿈(newline)
\r캐리지 리턴(carriage return)
\t수평 탭(tab)
\0널 문자(null)

문자열 상수​

  • 이중 인용부호(””)로 묶여 있는 복수개의 영문자나 숫자
  • 기억공간에 보관될 때는 문자열 끝에 null 문자(\0)가 추가
    • 예를 들어 문자열 “SEOUL KOREA”의 기억보관 형태

특수 상수​

특수 상수설명
참(True)불리언 자료형의 참을 나타내는 상수
거짓(False)불리언 자료형의 거짓을 나타내는 상수
NULL 또는 널포인터 자료형에서 사용되며, 아무런 값도 가리키지 않음을 나타냄
Infinity부동 소수점 자료형에서 양의 무한대를 나타내는 상수
NaN부동 소수점 자료형에서 숫자가 아님을 나타내는 상수
PI수학 상수로서, 원주율을 나타내는 상수
EOF파일 처리에서 파일의 끝을 나타내는 상수

변수​

  • 변할 수 있는 값
  • 프로그램에서 변수는 다음과 같이 정의됩니다:
    • 프로그램 실행 도중 변할 수 있는 값이 저장되는 기억공간을 의미합니다.
    • 예를 들어, I = 10;은 다음과 같이 해석됩니다
      • I는 변수(명)이고, 10이란 값을 I라는 이름으로 정의된 기억공간에 저장한다는 의미입니다.
  • 변수에 저장된 값은 수시로 변경될 수 있습니다.
  • 따라서 변수는 사용 전에 반드시 선언하여 컴파일러가 기억공간에서 일정 공간을 확보할 수 있도록 해야 합니다.

변수의 특징​

  • 이름이 있다.(변수명)
  • 정해진 자료형이 있다.
  • 할당된 값을 갖는다.

변수명의 정의 규칙​

  • 모든 변수는 사용되기 전에 선언되어야 합니다.
  • 변수명은 반드시 영문자나 밑줄(_)로 시작해야 합니다.
  • 변수명은 중간에 숫자, 밑줄(_)을 섞어서 명명할 수 있습니다.
  • 변수명은 중간에 밑줄(_) 이외의 특수문자($, #, ?, ...)를 포함할 수 없습니다.
  • 변수명은 대소문자를 구분하여 사용합니다.

선행처리기​

컴파일에 앞서 프로그램 선두에 선언된 지시자들을 미리 처리하는 역할을 수행

선행처리기설명
#include다른 파일의 내용을 현재 파일에 포함시킵니다.
#define상수나 매크로를 정의합니다.
#if조건을 검사하고 조건이 참이면 해당 코드를 컴파일합니다.
#else이전의 조건이 거짓이라면 해당 코드를 컴파일합니다.
#elif#if나 #ifdef 이후의 조건이 거짓이라면 새로운 조건을 검사합니다.
#endif조건부 컴파일 블록을 종료합니다.

선행처리기를 사용할 때 주의할 점

  • 반드시 #로 시작
  • 명령문 끝에는 세미콜론(;)을 붙이지 않는다.
  • 한 줄에 하나의 명령만 쓴다.
  • 소스 프로그램의 첫 부분에 위치한다.

#include

  • #include는 C 언어에서 다른 파일의 내용을 현재 파일에 포함시킬 때 사용됩니다.
  • 주로 표준 라이브러리나 사용자가 작성한 헤더 파일을 포함하기 위해 사용됩니다.
  • 예를 들어, 표준 입력과 출력을 지원하는 stdio.h 헤더 파일을 포함하기 위해서는 #include <stdio.h>와 같이 사용합니다.

#define

  • #define은 매크로를 정의할 때 사용됩니다.
  • 매크로는 프로그램 작성 시에 자주 사용되는 값이나 코드 조각을 미리 정의하여 사용할 수 있도록 해줍니다.
  • 매크로는 상수나 함수처럼 동작할 수 있으며, 선행처리기에 의해 컴파일 전에 해당 코드가 대체됩니다.
  • 매크로 정의에는 매크로 상수 정의와 매크로 함수 정의 두 가지가 있습니다.

매크로 상수 정의
#define 지시문을 사용하여 상수를 정의

#define PI 3.1415926535

매크로 함수 정의
#define 지시문을 사용하여 함수처럼 동작하는 코드 조각을 정의하는 것

#define SQUARE(x) ((x) * (x))

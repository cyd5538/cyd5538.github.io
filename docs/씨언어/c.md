# C언어란?

## C언어
C언어는 1972년 데니스 리치가 개발한 고급 프로그래밍 언어입니다.
> **프로그래밍 언어란** ✔  
> 사람과 컴파일러가 이해할 수 있도록 약속된 형태의 언어를 의미합니다.
> 기계어, 어셈블리어, C, C++, C#, Java, PHP 등과 같은 다양한 형태로 나타납니다.

> **컴파일러** 프로그래밍 언어로 작성된 프로그램을 기계어로 바꿔주는 번역기  
> **어셈블러** 기호로 표현된 어셈블리 코드를 기계어로 변역하는 번역기  
> **인터프리티** 소스 프로그램을 한번에 기계어로 변환시키는 컴파일러와는 달리 프로그램을 한 단계씩 기계어로 해석하여 실행하는 언어처리 프로그램(JS)

## C언어의 역사
- 1972년 데니스 리치가 AT&T 벨 연구소에서 개발
- UNIX 운영체제 개발 과정에서 탄생
- 당시 사용되던 어셈블리 언어는 기종 간 호환성이 떨어지고 하드웨어 의존성이 높아 새로운 언어가 필요했습니다. C언어는 이러한 문제를 해결하고 운영체제 개발 효율성을 높이기 위해 개발되었습니다.

## C언어의 특징

- 높은 프로그램 이식성
  - C언어는 대부분의 운영 체제에서 지원되어 다양한 플랫폼에서 프로그램을 실행할 수 있습니다.
  - 이식성이 높다는 말은 동일한 소스 코드를 다양한 운영체제나 하드웨어 환경에서 컴파일하여 실행할 수 있다는 의미입니다. 

- 간결하고 명확한 문법
  - C언어는 다른 고급 언어에 비해 문법 규칙이 비교적 단순하고 명확합니다.
- 함축적인 프로그램 작성 가능
  - C언어는 간단한 표현으로 복잡한 기능을 구현할 수 있는 강력한 표현력을 가지고 있습니다.
- 저급 언어 특성을 가진 고급 언어
  - C언어는 고급언어의 편리함과 저급언어의 기능을 갖추고 있습니다. 
  - 저급언어의 주요기능으로는 직접 메모리 관여, 시스템 호출 등
  - 고급언어의 주요기능으로는 간결한 문법, 라이브러리 지원, 프로그래밍 지원 등

> 저급언어는 컴퓨터가 쉽게 알아들을 수 있는 언어, 고급언어는 컴퓨터가 잘 알아듣기 어려운 언어

## C프로그램 구성

### 기본 구조
```c
#include <stdio.h> // 표준 입출력 라이브러리를 프로그램에 포함하는 지시문

int main() { // 가장 먼저 실행되는 함수 모든 프로그램에 반드시 존재
    myFunction();
    return 0;
}

void myFunction() { // main 함수에서 호출되는 함수 
    printf("Hello, world!\n");
}
```
- C 언어는 하나 이상의 함수를 포함해야 합니다.
- 프로그램의 실행이 시작되는 main() 함수는 반드시 존재해야 합니다.
- 함수는 중괄호 {}로 둘러싸여야 합니다.
- 중괄호 안에는 변수 선언문, 치환문, 연산문, 함수 등의 명령을 기입할 수 있습니다.
- 문장의 끝에는 세미콜론 ;을 붙입니다.

### 예약어
C 언어에서 특별한 의미를 가지는 단어입니다.
- 자료형 관련 예약어
  - char, int, float, short, long, double, unsigned, union, enum, void, ...
- 기억 관련 예약어
  - auto, static, extern, register, ...
- 제어 관련 예약어
  - if-else, for, while, do~while, switch-case, break, continue, return, ...
- 기타 예약어
  - main, sizeof, include, ...

### 명칭
명칭은 변수, 함수, 배열 등의 요소들에 부여되는 이름을 말합니다. 
- 첫 글자: 영문자(대소문자 구분), 숫자, _ 사용 가능
- 그 이후 글자: 영문자(대소문자 구분), 숫자, _ 사용 가능
- 키워드 사용 불가: 예약어는 사용할 수 없음
- 공백 사용 불가: 띄어쓰기 없이 작성해야 함
- 특수문자 사용 제한: 일부 특수문자는 사용 가능하지만 권장하지 않음

### 상수
프로그램 실행 중 변경되지 않는 값입니다.

### 연산자
- 산술 연산자, 논리 연산자, 비교 연산자 등 다양한 종류
- `+, -, *, /, &&, ||, ==, !=, <, >, <=, >=`

### 주석
- 프로그램 코드에 대한 설명을 작성하는 문장입니다.
- 여러 줄에 걸친 설명문을 넣기 위해서는 /* 와 */을 사용합니다.
- 한 줄 짜리 설명문을 넣기 위해서는 보통 //를 사용합니다.
  
### 에러
- C 언어 문법 또는 규칙에 어긋나는 경우 에러 메시지를 출력합니다.
- 에러 메세지를 확인하여 반드시 수정하여야 한다.

### 경고
- 잠재적인 에러? 경미한 실수를 했을때 발생합니다.
- 프로그램 실행을 방해하지는 않지만, 문제 해결을 권장합니다.

# 함수와 연산자

## 함수란?
함수는 특정한 작업을 수행하도록 설계된 독립적인 프로그램이라고 생각하면 됩니다.
함수는 코드를 모듈화하여 재사용성을 높이고, 복잡한 작업을 단순화하는 데 도움이 됩니다. 함수를 사용하면 비슷한 작업을 반복하는 대신 한 번 작성한 코드를 여러 곳에서 호출하여 사용할 수 있으므로 코드의 효율성과 가독성을 향상시킬 수 있습니다. 

## 표준 함수
사용자가 함수를 정의 할 수도 있는데 C언어에서는 자체적으로 제공하는 함수들이 있습니다 이를 표준 함수라고 합니다.

### 표준 출력 함수

| 함수         | 기능                                                              |
|-------------|--------------------------------------------------------------------|
| `printf()`  | 화면에 여러 종류의 데이터를 출력                                     |
| `putchar()` | 화면에 단일 문자를 출력                                             |
| `puts()`    | 화면에 문자열을 출력                                                |

**사용 예시**
```c
int main() {
    int num = 10;
    char ch = 'A';
    char str[] = "Hello, World!";

    printf("%s\n", str);

    putchar(ch);
    putchar('\n'); 

    puts(str);
}

```
`출력`
```
Hello, World!
A
Hello, World!
```
**출력양식 변환기호**
| 변환 기호 | 설명                                   | 예시                     |
|-----------|----------------------------------------|--------------------------|
| `%c`      | 단일 문자 출력                         | `printf("%c", 'A');`    |
| `%d`      | 부호 있는 10진수 정수 출력             | `printf("%d", 10);`     |
| `%e`, `%E`| 지수 형식의 부동 소수점 수 출력       | `printf("%e", 3.14);`   |
| `%f`      | 부동 소수점 수 출력                    | `printf("%f", 3.14);`   |
| `%g`, `%G`| 값에 따라 `%e` 또는 `%f` 서식 사용    | `printf("%g", 3.14);`   |
| `%o`      | 8진수 정수 출력                        | `printf("%o", 10);`     |
| `%s`      | 문자열 출력                             | `printf("%s", "Hello");`|
| `%u`      | 부호 없는 10진수 정수 출력             | `printf("%u", 10);`     |
| `%x`, `%X`| 16진수 정수 출력                       | `printf("%x", 255);`    |
| `%%`      | '%' 문자 출력                          | `printf("100%%");`      |

`출력양식 예`
```c
#include <stdio.h>

int main() {
    // 정수를 출력하는 예시
    printf("|%d|\n", 123);  // |123|

    // 필드 폭을 지정하여 정수를 출력하는 예시
    printf("|%5d|\n", 123); // |  123|

    // 좌측 정렬하여 정수를 출력하는 예시
    printf("|%-5d|\n", 123); // |123  |

    // 필드 폭을 지정하고 남는 공간을 0으로 채워 정수를 출력하는 예시
    printf("|%05d|\n", 123); // |00123|

    // 소수점 이하 첫 번째 자리까지 표시하고 필드 폭을 지정하여 부동 소수점 수를 출력하는 예시
    printf("|%06.1f|\n", 123.45); // |0123.5|

    // 필드 폭을 지정하고 소수점 이하 72자리까지 표시하는 예시
    printf("|%0.72f|\n", 123.45); // |123.450000000000002842170943040400743484497070312500000000000000000000000000|
 
}
```

### 표준 입력 함수
| 함수        | 설명                                          | 예시                        |
|-------------|-----------------------------------------------|-----------------------------|
| `scanf()`   | 여러 종류의 자료를 키보드로부터 입력 받음    | `scanf("%d", &num);`       |
| `getchar()` | 1개의 문자를 키보드로부터 입력 받음         | `ch = getchar();`           |
| `gets()`    | 문자열을 키보드로부터 입력 받음              | `gets(str);`       |

**입력양식 변환기호**
| 변환 기호 | 설명                    | 예시                   |
|-----------|-------------------------|------------------------|
| `%d`      | 정수형 입력 받음        | `scanf("%d", &num);`  |
| `%c`      | 문자형 입력 받음        | `scanf("%c", &ch);`   |
| `%s`      | 문자열 입력 받음        | `scanf("%s", str);`   |
| `%f`      | 실수형 입력 받음        | `scanf("%f", &float_num);`   |
| `%lf`     | double 실수형 입력 받음 | `scanf("%lf", &double_num);` |
| `%u`      | 부호 없는 10진수 정수 입력 받음 | `scanf("%u", &num);` |
| `%x`      | 16진수 정수 입력 받음   | `scanf("%x", &num);`   |

## 연산자
### 산술 연산자
| 연산자     | 설명          | 예시        | 연산자 | 설명 | 예시 |
|------------|-------------|-------------|------------|-------------|-------------|
| 이항       | 덧셈         | `a + b`     |단항       | 양수         | `+a`        |
|            | 뺄셈         | `a - b`     |          | 음수         | `-a`        |
|            | 곱셈         | `a * b`     |          | 증가         | `++a`       |
|            | 나눗셈       | `a / b`     |          | 증가         | `a++`       |
|            | 나머지       | `a % b`     |          | 감소         | `--a`       |
|            |              |             |          | 감소         | `a--`       |

**단항 연산자 예시**

```c
#include <stdio.h>

int main() {
    int a = 5;
    int b;

    b = a++; // b에 현재 a값을 할당하고 a가 1 증가
    printf("a = %d, b = %d\n", a, b); // a = 6, b = 5

    b = --a; // a의 값을 -1 감소하고 b에 할당
    printf("a = %d, b = %d\n", a, b); // a = 5, b = 5
}
``` 
 
### 관계 연산자

| 연산자           | 설명             | 예시            |
|------------------|------------------|----------------|
| `== `              | 같음을 비교       | `a == b`       |
| `!=`               | 같지 않음을 비교  | `a != b`       |
| `>`,`<`, `<=`, `>=`| 대,소관계를 비교  | `a > b` `a<=b` |

### 논리 연산자

| 연산자 | 설명                         | 예시            |
|--------|-----------------------------|-----------------|
| &&     | 양쪽 모두 true면 true        | `a && b`        |
| \|\|   | 양쪽 중 하나라도 true면 true | `a \|\| b`      |
| !      | 부정                        | `!a`            |

### 대입 연산자

| 연산자 | 설명                                                       | 예시            |
|--------|------------------------------------------------------------|-----------------|
| `=`      | b를 a에 할당합니다.                                     | `a = b`         |
| `+=`     | b를 a에 더하여 할당합니다.                               | `a += b`        |
| `-=`     | b를 a에서 빼고 할당합니다.                               | `a -= b`        |
| `*=`     | b를 a에 곱하여 할당합니다.                               | `a *= b`        |
| `/=`     | a를 b로 나눈 값을 a에 할당합니다.                        | `a /= b`        |
| `%=`     | a를 b로 나눈 나머지를 a에 할당합니다.                    | `a %= b`        |
| `<<=`    | a를 b 비트만큼 왼쪽으로 시프트하여 할당합니다.            | `a <<= b`       |
| `>>=`    | a를 b 비트만큼 오른쪽으로 시프트하여 할당합니다.          | `a >>= b`       |
| `&=`     | a와 b의 비트 AND 연산 결과를 a에 할당합니다.             | `a &= b`        |
| `\|=`    | a와 b의 비트 OR 연산 결과를 a에 할당합니다.              | `a \|= b`       |
| `^=`     | a와 b의 비트 XOR 연산 결과를 a에 할당합니다.             | `a ^= b`        |

### 조건 연산자
```c
#include <stdio.h>

int main() {
    int a = 5;
    int b = 10;
    int result;

    result = (a > b) ? a : b;

    printf("더 큰 값은: %d\n", result); //  더 큰 값은: 10
}
```
### 비트 연산자
| 연산자 | 설명                                         | 예시            |
|--------|---------------------------------------------|-----------------|
| `&`      | 대응되는 두 bit가 모두 1일 때만 결과는 1      | `a & b`         |
| `^`      | 대응되는 두 bit가 서로 다를 때만 결과는 1     | `a ^ b`         |
| `~`      | 1은 0으로,0은 1로 함                         | `~a`            |
| `<<`     | a의 값에 대해 2비트 왼쪽으로 이동             | `a << b`        |
| `>>`     | a의 값에 대해 2비트 오른쪽으로 이동           | `a >> b`        |

### 기타 연산자
| 연산자 | 설명                                             | 예시                    |
|--------|--------------------------------------------------|-------------------------|
| sizeof | 지정한 자료형, 수식, 변수가 차지하는 기억공간의 크기(byte)를 구합니다. | `sizeof(int)` 또는 `sizeof x` |
| cast   | 지정한 자료형을 다른 자료형으로 강제적으로 바꿉니다. | `(float)x`             |
| &      | 주소 연산자로서 피 연산자의 주소를 나타냅니다.   | `&x`                   |
| *      | 내용 연산자로서 피 연산자의 내용을 가져옵니다.   | `*ptr`                 |
